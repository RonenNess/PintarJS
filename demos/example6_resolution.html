Example 6: Resolution

PintarJS can help you maintain a constant rendering resolution. Due to browser limitations and the fact that users can control the window size, you can't really set both X and Y resolution, you must either set resolution width or height, and PintarJS will adjust the other axis accordingly to maintain a 1:1 ratio.

In this example, we lock width to 800px resolution, which means our rendering width will always be 800, and height will vary based on browser size.
